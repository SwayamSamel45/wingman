## Wingman
Wingman is an AI-powered texting assistant that helps you craft engaging, witty, and confident responses. Simply upload a screenshot of your conversation, and Wingman analyzes the chat context, tone, and flow to generate reply suggestions that keep the conversation going. Whether you're trying to be funny, flirty, confident, or smooth, Wingman helps you find the right words.

## Features
📸 Screenshot-based conversation analysis
🤖 Context-aware AI reply generation
😏 Flirty and "rizz" style suggestions
💬 Maintains conversation flow and tone
⚡ Fast and intuitive user experience
🧠 Uses modern Generative AI techniques for response generation

## How It Works
1. Upload a screenshot of your conversation.
2. Wingman extracts and analyzes the chat context.
3. The AI understands:
4. Conversation history
5. Tone and sentiment
6. User intent
7. Response opportunities
8. Wingman generates multiple suggested replies.
9. Choose the response that best matches your style.

## TechStack
1. Python
2. Groq API
3. OCR for Screenshot Text Extraction
4. Frontend Interface

## Why I built this
Texting can sometimes feel awkward, especially when you're trying to keep a conversation interesting. Wingman was built to bridge that gap by combining screenshot understanding with Generative AI to help users respond with confidence and personality. The goal wasn't to automate conversations, but to help users communicate better.
